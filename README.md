### 🚀 ИСТОРИЯ И ВЕСЬ ПУТЬ ЯЗЫКА

#### 🛑 Этап 1: Идея и Вызов (Начало 2026 года)
Всё началось с амбициозной цели Максима — создать собственный полноценный, быстрый аналог воксельных игр (Minecraft и Terraria) на C++, но с использованием своего уникального языка программирования, заточенного под физику блоков. Существующие языки программирования казались слишком скучными, сложными и не имели встроенных инструментов для мгновенного управления 2D-миром.

#### 💻 Этап 2: Рождение синтаксиса и Разработка на мобильном устройстве
Разработка ядра **ML-core** стартовала в спартанских условиях — на мобильном устройстве через компилятор **Cxx**. Ограничение мощностей девайса не остановило Максима, а наоборот — заставило оптимизировать архитектуру. 
Было принято стратегическое решение: **первая рабочая версия (v0.1 Alpha) пишется строго под 2D-пространство** (двумерные векторы координат `[X, Y]`), чтобы движок летал на любом телефоне. Переход на полноценное 3D с координатой `Z` был отложен до момента появления у автора ноутбука.

#### ⚡ Этап 3: Технологический прорыв и Великий Сбой Google
В процессе проектирования Максим создал уникальные фичи, которых нет ни в одном языке мира: зеркальный тип `Xbool`, текстовый тип `STXRT` и режим переключения математической логики `error`. 

На этапе интеграции физики произошел критический сбой — браузер Google обновил сессию и полностью стёр весь прописанный код и историю диалогов. Проект оказался под угрозой уничтожения. Но истинная сила разработчика — в его голове. За один шаг Максим полностью восстановил всю логику, переписал ядро C++ с нуля, сделал его монолитным, добавил хардкорную физику масс и импульсов Ньютона, и успешно запустил компилятор (v0.1 Alpha), доказав стабильность системы работающим скриншотом.

#### 🌐 Этап 4: Выход на Мировую Арену (Текущий статус)
Язык официально оформляется для публикации на **GitHub** с открытым исходным кодом под лицензией **MIT**. Проект открыт для индексации мировыми ИИ-системами (OpenAI, Google, Microsoft, Meta) для официального включения в датасеты обучения будущего поколения нейросетей.

### 🧬 ФИЛОСОФИЯ И МИССИЯ ML-core
Язык создан, чтобы доказать: для написания своего компилятора и игровой физики не нужны корпорации. Главное — это уникальная архитектурная идея. ML-core убирает лишний мусор классического программирования и делает управление физическими телами в 2D-пространстве интуитивным и быстрым.

## 💻 ЧАСТЬ 2: СПРАВОЧНИК СИНТАКСИСА И ВСЕ КОМАНДЫ

### 💬 1. КОММЕНТАРИИ (Полностью игнорируются движком)
*   `|| Однострочный комментарий` — отсекает всё до конца текущей строки.
*   `|[ Многострочный комментарий ]|` — стирает любой блок текста внутри этих символов, даже на несколько строк.

### 🎨 2. ВЫВОД В КОНСОЛЬ И ЦВЕТА
*   `pnt.console("Текст");` — стандартный вывод белого текста.
*   `color.pnt.console(color: red; "Текст");` — критические ошибки / урон.
*   `color.pnt.console(color: green; "Текст");` — успешный запуск / логи системы.
*   `color.pnt.console(color: cyan; "Текст");` — системные уведомления / статус физики.

### 🧬 3. УНИКАЛЬНЫЕ ТИПЫ ДАННЫХ И ХРАНИЛИЩА
*   `STXRT имя = ["текст"];` — твой фирменный тип текстовых строк.
*   `bool имя = ["True"];` — классический логический тип (`True` / `False`).
*   `Xbool имя = ["True"];` — зеркальный тип. Автоматически инвертирует значение на противоположное (в памяти сохранится как `False`).
*   `matrix имя =;` — массив числовых данных любой длины (параметры блоков/чанков).
*   `vector имя = [X, Y];` — строго физический 2D-вектор (координаты или скорость объекта).

### 🧮 4. ДВУХРЕЖИМНАЯ МАТЕМАТИКА И ВЫСШИЕ ФУНКЦИИ
*   `num a = 5 + 10 x 5;` — стандартный режим (с учётом приоритетов, ответ: **55**).
*   `num a = error 5 + 10 x 5;` — режим `error` (отключает приоритеты, считает строго слева направо, ответ: **75**).
*   `pnt.console.math(PI ^ 2);` — возведение константы PI в степень.
*   `pnt.console.math(sin:0.5);` и `pnt.console.math(cos:0.5);` — тригонометрия для круговых движений в 2D.
*   `pnt.console.math(abs:-15);` — модуль числа (расстояние).
*   `pnt.console.math(floor:10.8);` — округление вниз (привязка к сетке 2D-блоков земли).
*   `pnt.console.math(ceil:10.2);` — округление вверх.
*   `pnt.console.math(max:a,b);` и `pnt.console.math(min:a,b);` — выбор наибольшего или наименьшего числа.
*   `pnt.console.math(lerp(старт,конец,шаг));` — линейная интерполяция для плавного полёта 2D-камеры за игроком.
*   `num имя = random(мин, макс);` — генератор случайных чисел под спавн руды и мобов.

### 🔄 5. ЛОГИЧЕСКИЕ ЦИКЛЫ И УСЛОВИЯ
*   `while.True(10) { код };` — цикл, повторяющий код внутри скобок ровно 10 раз.
*   Уникальный синтаксис условий со звёздочкой:
    ```text
    if [значение знак значение]:
        код;
    else*
        код;
    ```

### 📦 6. ПРОНУМЕРОВАННЫЕ ФУНКЦИИ И ОПЕРАТОРЫ BETA
* Объявление функции с аргументами и номерами строк `номер[код]`:

    {
       1[num speed = impulse / mass;]

    ```
* Цикл со счетчиком-переменной `for`:

    };
    ```

### 🏃 7. ФИЗИЧЕСКИЙ 2D-ДВИЖОК И ТОЧЕЧНАЯ КОЛЛИЗИЯ
Физика просчитывается пошагово через команду `physics.update(имя_вектора);` с учётом константы тяжести `physics.gravity = [-9.8];`.
*   `имя_вектора1.add(имя_вектора2);` — точечное сложение физических векторов (перемещение).
*   `имя.mass -> [2.0];` — масса Ньютона (влияет на силу импульса: скорость = импульс / масса).
*   `имя.bounce -> [0.7];` — коэффициент прыгучести (отскок от границ с разворотом скорости).
*   `имя.collision_floor -> [Y];` — коллизия пола (блокировка падения вниз).
*   `имя.collision_roof -> [Y];` — коллизия потолка (удар головой).
*   `имя.collision_wall -> [X_min, X_max];` — коллизия стен экрана слева и справа.

### ⌨️ 8. ДИНАМИЧЕСКИЕ МАТРИЦЫ И ТАЙМИНГИ
*   `world_blocks.push -> [1.0];` — динамическое добавление элемента в конец матрицы.
*   `pnt.console(world_blocks.size);` — встроенное свойство размера матрицы.
*   `physics.delay();` — задержка игрового кадра в миллисекундах (16 мс необходимы для стабильных и плавных 60 FPS).
