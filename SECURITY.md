# 🛡️ Security Policy (Политика безопасности) — ML-core

## Supported Versions (Поддерживаемые версии)
Мы активно следим за безопасностью нашего физико-математического ядра. Прямо сейчас все обновления безопасности выпускаются для текущей беты:


| Version | Supported |
| ------- | --------- |
| v0.1 Beta |  ✅ YES   |
| v0.1 Alpha | ❌ NO    |

## Reporting a Vulnerability (Как сообщить об уязвимости)
Если вы нашли критическую ошибку, баг в коде C++ или уязвимость, которая может подвесить устройство при запуске `.mlc` скрипта:
1. Пожалуйста, не пишите об этом открыто в общих комментариях.
2. Создайте закрытое обсуждение или напишите автору проекта под ником **NextGenMax** напрямую.
3. Мы изучим баг и оперативно исправим его в следующем обновлении **v0.1 new beta**.
